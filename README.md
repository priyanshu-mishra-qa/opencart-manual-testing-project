# OpenCart Manual Testing Project

This repository contains a manual testing portfolio project for the OpenCart Demo Store e-commerce application.

## Scope of Testing
- Registration
- Login / Logout
- Forgot Password
- Search
- Product Display
- Cart
- Checkout

## Test Artifacts Included
- Test Plan
- Test Scenarios & Test Cases
- Requirement Traceability Matrix (RTM)
- Test Execution Report
- Defect Log
- Test Summary Report

## Testing Summary
Total Test Cases: 99  
Pass Rate: 82%  
Defects Identified: 11  

This project demonstrates end-to-end manual testing documentation and defect tracking for an e-commerce application.
